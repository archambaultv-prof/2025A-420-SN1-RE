# Examen théorique intra

- Sur papier.
- L'examen couvre toute la matière incluant listes et matplotlib (semaine 6).
- L'examen dure 3h00.
- L'examen ne comportera aucune question à choix multiple. Les questions seront plutôt dans les styles suivants :
    - Qu'affiche ce code?
    - Compléter ce code pour qu'il fasse telle chose.
    - Écrire un programme (court) pour faire telle chose.
- Pas d'algorithmes dans cet examen.
- Pas d'accès aux environnements de développement (ni ALIVECode, ni PyCharm, etc.).
- La dernière page de l'examen sera un [aide-mémoire](pathname:///files/Annexe_Examen_v1.docx).
- Aucun document n'est autorisé, sauf l'aide-mémoire fourni.
